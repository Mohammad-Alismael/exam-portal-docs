# Create User

User creation is a fundamental process in any system or application that involves registering new users or accounts. It allows individuals to gain access to various features, services, or functionalities offered by the platform.
### **API Endpoint**

```http
  POST /users/create
```

#### Request body object

| Parameter  | Type     | Description                                 |
| :--------- | :------- | :------------------------------------------ |
| `username` | `string` | **Required**. username feild must be filled |
| `password` | `string` | **Required**. password feild must be filled |
| `email_id` | `string` | **Required**. email id feild must be filled |
| `role id` | `integer` | **Required**. role id feild must be filled, read more about [user types](http://localhost:3000/documentation/users/user-types) |

#### **Request body**

```json
{
    "username": "user",
    "password": "123",
    "email_id": "user@gmail.com",
    "role_id": 1
}
```
