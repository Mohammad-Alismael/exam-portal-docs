# Reset Password

Users can reset their passwords by providing their email address. When a user submits their email, our system will send
a reset password email to the provided email address. This email will contain a secure link that redirects the user back
to our system's password reset page. On the password reset page, the user can provide a new password. Once the new password
is submitted, our system will update the user's password accordingly, ensuring their account remains secure.

### **API Endpoint**

```http
  POST /users/password/reset
```


#### Request body object

| Parameter     | Type     | Description                                                                    |
| :------------ | :------- | :----------------------------------------------------------------------------- |
| `reset_token` | `string` | **Required**. reset token will be sent via an email |
| `new_password` | `string` | **Required**. new passsword that the user wants to change to |

#### **Request body**

```json
{
    "reset_token" : "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJ1c2VyX2lkIjo1LCJpYXQiOjE2OTE1NzQxODAsImV4cCI6MTY5MTU3NTk4MH0.2ggdmlPnfIprFfrlIKyIHPNssk-oN9-KAeQ66kbQ470",
    "new_password": "123456"
}
```

#### **Response body**

```json
{
    "message": "password updated successfully!"
}
```
